# apuntes de física - 2º bachillerato

aquí dejo mis apuntes de física para el curso de 2º de bachillerato. escritos en latex y ya están compilados para que puedas descargarte el PDF desde el código (es el documento `document.pdf`). también puedes editarlos y compilarlos por tí mismo si tienes latex instalado en tu ordenador. el código está estructurado de la siguiente forma:

- documento `document.tex`: el documento principal, que contiene el código de la portada y el índice.
- directorio `SECTIONS/`: directorio que contiene las secciones del documento, cada una en su propio archivo .tex.
- directorio `FIGURES/`: directorio que contiene las figuras del documento, cada una en su propio archivo .png.

dentro de `SECTIONS/` están todos los temas principales de PAU:

- [x] `1 - GRAV.tex`: gravedad y campo gravitatorio.
- [x] `2 - ELECT.tex`: electricidad y campo eléctrico.
- [x] `3 - MAGN.tex`: magnetismo y campo magnético.
- [x] `4 - INDUCC.tex`: inducción electromagnética.
- [x] `5 - ONDAS-SONIDO.tex`: ondas mecánicas como las de sonido.
- [ ] `6 - ONDAS-EM.tex`: ondas electromagnéticas.
- [ ] `7 - OPT.tex`: óptica geométrica.
- [ ] `8 - RELAT.tex`: relatividad y la _nueva física_.
- [ ] `9 - FISCUANT.tex`: física cuántica.
- [ ] `10 - FISNUC.tex`: física nuclear.
